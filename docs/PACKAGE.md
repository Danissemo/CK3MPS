# GitHub Package

CK3MPS is published to GitHub Packages as a NuGet package for repository visibility and archival.

Main user download:

```text
https://github.com/Danissemo/CK3MPS/releases/latest
```

Package page:

```text
https://github.com/Danissemo/CK3MPS/pkgs/nuget/CK3MPS
```

The package contains:

```text
tools/CK3MPS.exe
tools/README.md
README.md
CHANGELOG.md
LICENSE
```

For normal use, download `CK3MPS.exe` from Releases. The NuGet package is not intended as the primary end-user installer.
