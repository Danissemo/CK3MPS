# Screenshots

Current repository screenshots live in `assets/screenshots`.

## Required Files

- `main-window.png`
- `scan.png`
- `report.png`
- `restore.png`

## Update Rules

1. Rebuild the official `release\CK3MPS.exe` first.
2. Capture screenshots only from the current release candidate UI.
3. Replace existing files instead of creating version-suffixed screenshot names.
4. Remove stale screenshot files after renames or UI workflow changes.
5. Check `README.md` references after updating file names.

## When To Refresh

- Main tab layout changed
- Reports tab layout changed
- Restore tab layout changed
- Button names or workflow labels changed
- Release branding or version framing changed in a way visible on the screenshots
