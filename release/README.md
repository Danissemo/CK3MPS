# CK3MPS Release

```text
CK3MPS.exe
```

Run `CK3MPS.exe` as administrator.

Recommended first run:

1. Close CK3 and Paradox Launcher.
2. Select `Recommended`.
3. Click `Stabilize CK3`.
4. Run `Check Only`.

CK3MPS writes compact reports to:

```text
Documents\Paradox Interactive\CK3MPS
```

If you enable `Portable mode`, CK3MPS instead keeps its working files next to the executable in:

```text
CK3MPS_Data
```
